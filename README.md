# 🎬 YouTube视频管理工具

一个专业的纯前端YouTube视频管理工具，支持视频搜索、批量管理、评论获取和数据导出。

## ✨ 核心特性

### 🔍 智能搜索
- **关键词搜索** - 输入主题关键词，快速找到相关视频
- **URL解析** - 直接粘贴YouTube链接，自动获取视频信息
- **批量导入** - 支持一次性导入多个视频链接

### 🎯 重点功能：一键复制所有URL
- **突出显示** - 主要操作区域的大按钮设计
- **智能复制** - 支持现代剪贴板API和备用方法
- **批量处理** - 一次性复制所有筛选后的视频链接
- **选择性复制** - 支持复制选中的特定视频链接

### 📊 数据管理
- **本地存储** - 使用IndexedDB，数据完全本地化
- **状态管理** - 视频处理状态可视化
- **智能筛选** - 按状态、标题、频道等条件筛选
- **多种排序** - 按时间、标题、观看数等排序

### 📤 灵活导出
- **多种格式** - 支持TXT、CSV、JSON格式导出
- **自定义导出** - 可选择导出字段和范围
- **即时下载** - 客户端生成文件，无需服务器

### 🎨 现代界面
- **响应式设计** - 完美适配桌面、平板、手机
- **暗色主题** - 现代化深色主题设计
- **直观操作** - 简洁的用户界面，易于使用
- **快捷键支持** - 键盘快捷键提高操作效率

## 🚀 快速开始

### 1. 获取API密钥

1. 访问 [Google Cloud Console](https://console.developers.google.com/)
2. 创建新项目或选择现有项目
3. 启用 **YouTube Data API v3**
4. 创建API密钥（无需OAuth，仅API密钥即可）

### 2. 启动工具

**Python启动（推荐）：**
```bash
python start.py
# 访问 http://localhost:8001
```

**直接打开：**
- 双击 `index.html` 文件即可使用

### 3. 使用工具

1. **配置API** - 在页面顶部输入您的YouTube API密钥
2. **测试连接** - 点击"测试API"确保连接正常
3. **开始使用** - 搜索视频或粘贴链接

## 📋 功能说明

### 🔍 搜索功能
- **关键词搜索**：输入任何主题相关的关键词
- **URL导入**：粘贴以下格式的链接
  - `https://www.youtube.com/watch?v=VIDEO_ID`
  - `https://youtu.be/VIDEO_ID`
  - `https://www.youtube.com/shorts/VIDEO_ID`
- **批量URL**：多个链接用换行符或逗号分隔

### 🎯 一键复制功能
- **一键复制所有URL**：复制当前筛选结果的所有视频链接
- **复制选中**：仅复制勾选的视频链接
- **智能复制**：自动适配不同浏览器的剪贴板API
- **格式化输出**：每行一个链接，便于粘贴使用

### 📊 数据筛选与导出
- **状态筛选**：待处理、处理中、已完成、失败
- **文本搜索**：按标题、频道名称搜索
- **排序选项**：时间、标题、观看数、点赞数
- **导出格式**：TXT（纯URL列表）、CSV（完整信息）、JSON（结构化数据）

## ⌨️ 快捷键

## ❓ 常见问题

### Q: 为什么需要YouTube API密钥？
A: YouTube API密钥用于访问YouTube的官方数据接口，获取视频信息。这是Google的要求，确保数据访问的合规性。

### Q: API密钥是否安全？
A: 是的，API密钥仅存储在您的浏览器本地，不会上传到任何服务器。

### Q: 工具是否免费？
A: 完全免费。YouTube API有每日免费配额，对于个人使用完全足够。

### Q: 数据存储在哪里？
A: 所有数据都存储在您的浏览器本地IndexedDB中，不会上传到外部服务器。

### Q: 支持哪些YouTube链接格式？
A: 支持所有标准的YouTube链接格式，包括常规视频、短视频(Shorts)和嵌入链接。
